# Old but Gold

Animated short film created in **Blender** for a **Computer Graphics** course.

## Content
- Final `.blend` file with all assets packed (no external resources required).

## Animation Description
A short animated film about an emotional and slightly ironic interaction between two phones, exploring contrast, movement, and resilience through silence and visual storytelling.

## Notes
- Open with Blender (version 5.0.1 recommended).
